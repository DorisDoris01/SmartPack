# SmartPack

Smart packing list app for travelers with weather integration.

## Weather API

SmartPack uses [Open-Meteo](https://open-meteo.com/) for weather forecasts.

### Why Open-Meteo?

- ✅ **16-day forecast** - covers even long trips
- ✅ **Completely free** - no API key required
- ✅ **No registration needed** - works out of the box
- ✅ **No rate limits** - unlimited requests for non-commercial use
- ✅ **High accuracy** - uses multiple weather models
- ✅ **Open source** - transparent and reliable

### Setup

**No setup required!** Open-Meteo works automatically without any API key or registration.

## Building the App

1. Open `SmartPack.xcodeproj` in Xcode
2. Select your development team in Signing & Capabilities
3. Connect your iPhone and select it as the target
4. Click Run (⌘R)

## Features

- Multi-day trip planning
- Weather-based packing suggestions
- Customizable packing lists
- Tag-based item generation
- Multi-language support (English/Chinese)
