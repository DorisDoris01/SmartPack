What Changed vs. What Stayed the Same

Unchanged (everything else)

= Navigation bar title and close button
= Date range section (pickers, day count)
= SectionCard wrapper, icon, and title for destination
= TextField appearance (font, placeholder, color)
= Tag sections (activity, occasion, config)
= Generate button style and position
= Overall scroll layout and spacing
= Trip model and schema (no migration)

New: Drill-Down Only

+ Suggestion dropdown appears below text field when typing
+ Each suggestion shows city name + region/country
+ Tapping a suggestion fills the field and saves coordinates
+ Dropdown disappears after selection or on submit
✓ "赣州" now finds the correct city
✓ "北京" returns the capital, not a village
✓ "龙目岛" and "巴厘岛" now work
✓ "Bali" / "Lombok" return the correct islands
